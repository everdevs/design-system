## Usage

A Page provides the main layout of a webpage. Under the hood it uses [grid-layout](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_Grid_Layout)
in its most simple form.

The Footer is never visible when the page-scroll is 0. This ensures that the Main area
will stretch to the end of the viewport even if the page is empty.

```
┏━━━━━━━━━━━━━━┓ ┏━━━━━━━━━━━━━━┓
┃    HEADER    ┃ ┃              ┃
┣━━━━━━━━━━━━━━┫ ┃              ┃
┃              ┃ ┃   VIEWPORT   ┃
┃     MAIN     ┃ ┃              ┃
┃              ┃ ┃              ┃
┣━━━━━━━━━━━━━━┫ ┗━━━━━━━━━━━━━━┛
┃    FOOTER    ┃
┗━━━━━━━━━━━━━━┛
```

## Simple

```jsx
function MyComponent() {
	return (
		<Page>
      <Header>HEADER</Header>
      <Main>MAIN</Main>
      <Footer>FOOTER</Footer>
  </Page>
	);
}
```

## Adding a fixed header

```jsx
const fixedHeader = {
	position: "fixed",
	top: 0,
	left: 0,
	right: 0,
	height: "inherit"
}

function FixedHeader({className, children}) {
	return <div className={className} style={fixedHeader}>{children}</div>
}

function MyComponent() {
	return (
		<Page>
      <Header><FixedHeader>HEADER</FixedHeader></Header>
      <Main>MAIN</Main>
      <Footer>FOOTER</Footer>
  </Page>
	);
}
```
